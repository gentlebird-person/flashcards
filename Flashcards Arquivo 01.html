Informática Básica

Hardware
Hardware é tudo aquilo que você pode tocar em um computador ou equipamento eletrônico. São as peças físicas que fazem o sistema funcionar. Se você consegue chutar, é hardware.
Software
Software é tudo aquilo que você não pode tocar, mas que faz o computador funcionar. São os programas e sistemas que dizem ao hardware o que fazer. É o que você não pode chutar, então só pode xingar.
Desktop (Máquina)
Um desktop é o computador de mesa, aquele que fica fixo em um local, geralmente usado em escritórios ou estações de trabalho.
Mobile (Dispositivo)
Um mobile é um aparelho eletrônico portátil que a gente pode levar no bolso ou na mão. Os exemplos mais comuns são celulares (smartphones) e tablets.
Navegador
Um navegador (ou browser, em inglês) é um programa usado para acessar a internet. Os navegadores mais conhecidos são Chrome, Firefox, Edge e Safari. Pense no navegador como uma porta de entrada para o mundo da internet. Você digita o endereço de um site (como www.ailos.coop.br) e ele mostra o conteúdo desse site na tela.
Acesso Remoto
Acesso remoto é quando alguém usa um computador ou celular para entrar em outro computador à distância, como se estivesse sentado na frente dele — mesmo estando em outro lugar, cidade ou até país.
Sistema Operacional
O sistema operacional é o coração do computador, celular ou tablet. Ele é o programa principal que faz tudo funcionar. Sem ele, o aparelho não liga direito, não abre programas e não faz nada. Exemplos são: Windows (usado na maioria dos computadores), Android (na maioria dos celulares), iOS (iPhones), Linux (usado principalmente em servidores).
Formatar
Formatação é o processo de limpar completamente um computador, celular ou outro dispositivo, apagando tudo o que tem nele, e deixando como se tivesse saído da fábrica.Formatar é como esvaziar a mesa inteira, jogar tudo fora e começar do zero, com tudo limpo e organizado.
Backup
Backup é uma cópia de segurança dos arquivos importantes. Serve para proteger os dados caso algo dê errado. Porém, é necessário tomar cuidado para não realizar backups de dados/programas maliciosos/infectados!
Antivírus
Um antivírus é um programa de proteção que fica instalado no computador ou celular para defender o aparelho contra ameaças.
ID
ID é a abreviação de "identificador". Ele é como um RG digital: um código único que serve para identificar uma pessoa, um sistema, um equipamento ou um acesso dentro da empresa ou de um sistema.

Ameaças (Threats)

Engenharia Social
Engenharia Social é quando alguém tenta enganar uma pessoa para conseguir informações ou acesso a sistemas, sem usar tecnologia, mas sim explorando a confiança, distração ou falta de atenção.
Vulnerabilidade
Fraqueza em um sistema que pode ser explorada por uma ameaça.
Risco
Possibilidade de perda ou dano quando uma ameaça explora uma vulnerabilidade.
Firewall
Sistema que monitora e controla o tráfego de rede.
Criptografia
Codificação de dados para impedir acesso não autorizado.

Engenharia Social

Engenharia Social
Manipulação psicológica para obter acesso a informações ou sistemas.
Phishing
Mensagens falsas que induzem o usuário a fornecer dados confidenciais.
Pretexting
Criação de cenários falsos para enganar e obter informações.
Baiting
Oferecimento de algo atrativo para induzir ações inseguras.
Tailgating
Acesso físico não autorizado seguindo alguém em área restrita.

Infraestrutura de TI

Autenticação
Verificação da identidade de um usuário.
2FA
Autenticação de dois fatores para maior segurança.
Antivírus
Software que detecta e remove ameaças.
Ambiente de Desenvolvimento
Local onde funcionalidades são criadas e testadas.
Ambiente de Homologação
Ambiente que simula produção para validação de funcionalidades.

Segurança de Redes

VPN
Rede privada virtual que criptografa a conexão.
IP
Identificador único de um dispositivo na rede.
DNS
Sistema que traduz nomes de domínio em endereços IP.
MitM
Ataque onde o invasor intercepta comunicação entre duas partes.
Segmentação de Rede
Divisão da rede para limitar o impacto de ataques.

Fraude Documental

Documentos Falsificados
Uso de documentos adulterados para obter vantagens.
Identidade Falsa
Criação de perfis falsos para abertura de contas.
Comprovantes Fraudulentos
Apresentação de comprovantes de renda ou residência falsos.
Assinatura Falsa
Falsificação de assinatura em contratos.
Uso indevido de documentos legítimos
Utilização de documentos reais sem autorização.

Fraude Interna

Desvio de Recursos
Funcionário que manipula processos para benefício próprio.
Conluio com Terceiros
Colaboração entre colaborador e fraudador externo.
Acesso Indevido
Uso de credenciais para acessar dados confidenciais.
Manipulação de Sistemas
Alteração de registros ou parâmetros para ocultar ações.
Violação de Políticas
Descumprimento de normas internas de segurança.

Fraude com Cartões

Clonagem de Cartão
Cópia dos dados do cartão para uso indevido.
Skimming
Uso de dispositivos para capturar dados em caixas eletrônicos.
Uso sem senha
Compras realizadas via NFC sem autenticação.
Troca de Cartão
Substituição do cartão da vítima por outro sem que ela perceba.
Falsa Central
Golpista se passa por atendente e coleta o cartão da vítima.

Golpes Digitais

Marketplace Falso
Sites falsos que simulam lojas legítimas.
Vazamento de Dados
Exposição de dados sensíveis em fóruns clandestinos.
Phishing Bancário
Simulação de páginas de bancos para roubo de credenciais.
Golpe do Boleto
Boleto com dados alterados para redirecionar o pagamento.
Golpe do Pix
QR Code ou chave Pix falsos para desviar valores.
